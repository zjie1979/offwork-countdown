# 下班倒计时

一个可公开部署到 GitHub Pages 的 iPhone PWA，用于显示起床、出门、上班、下班和摸鱼正计时。

## 功能

- 上班、下班和出门默认周一至周五计时。
- 默认起床时间 `07:30`，可在设置里自定义；起床倒计时可选择跟随计时日期或每天，从目标日前一晚 `22:00` 开始，到起床时间后停止。
- 默认上班出门时间 `08:50`，可在设置里改成 `08:30`、`08:35` 等时间；出门倒计时只从工作日当天 `07:00` 开始，到出门时间后停止。
- 默认上午上班 `09:30`，中午下班 `12:00`，下午上班 `14:00`，下午下班 `18:00`。
- 根据当前时间自动显示“上午上班 / 中午下班 / 下午上班 / 下午下班”的下一个目标。
- 底部三页：倒计时、摸鱼、设置。
- 可设置每日摸鱼额度，点击开始后自动正计时，暂停后保留今天累计时间。
- 支持屏幕常亮开关；浏览器支持时会使用 Screen Wake Lock API。
- 起床时间、起床日期、出门时间、下班时间、计时日期和摸鱼记录保存在本机浏览器，不上传服务器。

## 发布范围

本仓库只包含静态网页文件，不包含个人记录、账号、密钥或业务数据。
